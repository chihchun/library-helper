- [Kobo.com](https://www.kobo.com/tw/zh)

- [TAAZE 讀冊生活](https://www.taaze.tw/index.html)
- [TAAZE 讀冊生活二手中文書](https://www.taaze.tw/rwd_list.html?t=11&k=03&d=00)
